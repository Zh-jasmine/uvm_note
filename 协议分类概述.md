# 协议分类概述

## 片内总线（SoC 内部 IP 之间）

### AMBA 家族谱系

```
AMBA 1.0 ─── 已淘汰
├── ASB
└── APB

AMBA 2.0
├── AHB
└── APB

AMBA 3.0
├── AXI3
├── AHB
└── APB

AMBA 4.0
├── AXI4-Full      完整版，支持 burst
├── AXI4-Lite      精简版，不支持 burst，用于控制外设
├── AXI4-Stream    流式接口，无地址，专传数据流
├── ACE/ACE-Lite   缓存一致性扩展
├── AHB
└── APB

AMBA 5.0
├── CHI            多核缓存一致性，替代 ACE
├── AXI4
├── AXI4-Stream
└── APB
```

### 各总线定位

| 总线 | 定位 | 特点 |
|------|------|------|
| APB | 低速外设 | 最简单，仅 2 个控制信号，无流水 |
| AHB | 中高性能 | 比 AXI 简单，支持 burst，单通道 |
| AXI | 高性能 | 五通道分离，支持乱序、burst |
| CHI | 多核互联 | 缓存一致性，取代 ACE |

## 片间协议（芯片与外部设备之间）

### 串行类

| 协议 | 线数 | 特点 | 典型应用 |
|------|------|------|----------|
| SPI | 4 (MOSI/MISO/SCK/CS) | 全双工，速度快 | Flash、传感器、ADC |
| I2C | 2 (SCL/SDA) | 半双工，两线制，多设备 | 传感器、EEPROM、PMIC |
| UART | 2 (TX/RX) | 异步，无双钟线 | 调试串口、蓝牙、GPS |

### 并行/高速类

| 协议 | 分类 | 说明 |
|------|------|------|
| PCIe | 片间高速 | 接 GPU、SSD、网卡 |
| DDR | 片间内存接口 | CPU 接 DRAM 颗粒 |
| USB | 片间 | 键盘、U 盘、摄像头 |
| Ethernet | 片间 | 网络通信 |

## DUT 在协议体系中的位置

AXI4-Lite to SPI Bridge 是一个协议桥接器：

```
CPU
  │
  │ AXI4-Lite（片内总线）
  ▼
[ DUT ]
  │
  │ SPI（片间串行协议）
  ▼
SPI Slave 芯片（传感器、Flash、ADC...）
```

DUT 的 AXI 侧实现 AXI4-Lite Slave 接口，SPI 侧实现 SPI Master 功能。两个方向协议不同，DUT 负责在两者之间做协议转换和状态管理。
