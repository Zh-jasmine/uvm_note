# Markdown 速查

## 标题

```
# 一级
## 二级
### 三级
#### 四级
```

## 强调

```
**加粗**
*斜体*
~~删除线~~
==高亮==（Obsidian 支持）
```

## 列表

```
- 无序
- 无序
  - 缩进子项

1. 有序
2. 有序
```

## 链接

```
[文字](url)
[[笔记名]]         ← Obsidian 双链
[[笔记名|别名]]     ← 显示别名
```

## 代码

````
`行内代码`

```语言名
代码块
```
````

## 表格

```
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| a   | b   | c   |
```

对齐：`|:---|:---:|---:|`（左/中/右）

## 引用

```
> 引用
>> 嵌套
```

## 分割线

```
---
```

## 图片

```
![描述](路径)
```

## 任务列表

```
- [ ] 未完成
- [x] 已完成
```

## 脚注

```
文字[^1]

[^1]: 脚注内容
```

## 注释（Obsidian）

```
%% 这行不会渲染 %%
```

## 标签

```
#标签
#嵌套/标签
```

## Callout（Obsidian）

```
> [!note] 标题
> 内容

> [!warning] / [!tip] / [!info] / [!danger] / [!example]
```

## YAML Frontmatter

```
---
tags: [a, b]
date: 2026-04-28
---
```
